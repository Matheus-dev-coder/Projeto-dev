Gerador de Páginas com IA
📌 Sobre o Projeto

O Gerador de Páginas com IA é uma aplicação web que permite criar páginas de apresentação a partir de um simples prompt.

O usuário descreve seu negócio ou ideia (por exemplo: "cafeteria", "restaurante", "academia", "loja de roupas") e a aplicação utiliza Inteligência Artificial para gerar automaticamente uma página personalizada em segundos.

O objetivo do projeto é demonstrar como a IA pode acelerar o desenvolvimento de interfaces web e facilitar a criação de páginas para diferentes tipos de negócios.

🚀 Funcionalidades
Geração automática de páginas através de prompts.
Interface simples e intuitiva.
Layout responsivo.
Personalização baseada na descrição do usuário.
Integração com Inteligência Artificial.

🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript
API de Inteligência Artificial

💡 Exemplo de Uso

Digite um prompt como:

Cafeteria moderna com ambiente aconchegante.

Ou:

Academia focada em musculação e treinos funcionais.

A aplicação irá gerar uma página personalizada com base na descrição fornecida.
